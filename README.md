# Network Routing Simulator

A comprehensive C++ implementation of shortest path algorithms (Dijkstra, A*) for network routing with real-time visualization.

## Features

- **Multiple Algorithms**: Dijkstra's Algorithm and A* Search
- **Multiple Metrics**: Latency, Cost, and Reliability optimization
- **Real-time Visualization**: ASCII-based network topology with color coding
- **Step-by-step Execution**: Watch algorithms work in real-time
- **Performance Comparison**: Compare algorithm efficiency
